# document.addEventListener()とは  
特定の要素に対して特定のイベントが起こったときに、特定の動作（関数）を行うようにする  
対象要素.addEventListener( 種類, 関数, false )  
- 第1引数：イベントの種類　
- 第2引数：実行させる関数
- 第3引数：falseはイベントの順番を決めるもの。trueかfalseがあるが、通常はfalse。　

## 第３引数のイベントの順番　
- false（デフォルト） の場合：イベントが内側（子）から外側（親）に向かって伝わる　
- true の場合：イベントが外側（親）から内側（子）に向かって伝わる  
  これを**イベントの伝播**という　
<br>

# イベントの伝播とは？　
クリックなどのイベントが、HTMLの階層（親子関係）を上や下へ移動しながら伝わっていく仕組みのこと　

## イベント伝播には 3 つのフェーズがある  
① キャプチャリング（捕獲）フェーズ  
document → html → body → 親 → 子  
上から下へ イベントが降りてくる  

② ターゲット（目標）フェーズ  
実際にクリックされた要素（例：button）  

③ バブリング（泡のように上昇）フェーズ  👈🏻<u>**JavaScript のイベントは 基本はバブリング**</u>   
子 → 親 → body → html → document  
下から上へ 伝わって戻る  
<br>

### HTML 要素を使うときの注意点  
document.getElementById() で変数に入れてから使うのが良い    
　→なぜなら、DOM を1回しか探しに行かないので処理が速くなり、読みやすいコードになるから  
　スクロールやクリックで実行したい処理は関数にして渡しましょう！
