# モジュールとは  
**まとまった機能を1つの部品（ライブラリ）として提供するコードの集まり”**のこと。  
プログラムを 分割・再利用 するための仕組みである。  
外部ライブラリも、自分で作ったファイルもモジュール。  
import は その部品を使うための宣言。  

```  
import モジュール名
import モジュール名 as 別名    
```  
asを付けるとモジュール名を短く書けて便利になる。  
例：  
- numpy → np  
- pandas → pd  
- matplotlib.pyplot → plt  
<br>


## モジュールの役割  
① 機能をまとめて提供する。  
たとえば Python の requests モジュールは、  
- HTTPリクエスト  
- JSONのパース  
- エラー処理  
を全部まとめて提供してくれます。
<br>  
<br> 


② コードの分割・管理をしやすくする  
ファイル分割 ＝ モジュール化  
- user.py → ユーザー管理のコード  
- auth.py → 認証処理  
- database.py → DB接続  
のように、機能ごとにファイル（.py）を分けてそれをモジュールとして import します。  
<br>
<br> 


③ 他チーム・他サービスが作ったコードを使える  
- チームAが作った「共通ログ出力機能」  
- チームBが作った「ユーザー認証ロジック」  
などが、モジュールとして配布されて他の開発者が利用します。  
もちろん、自分でもモジュールを作れます。  
<br>
<br> 


④ 外部ライブラリやフレームワークもほぼモジュール  
- Python：Pandas、NumPy、Flask、Django  
- JavaScript：React、Axios、Lodash  
- Java：Spring、JUnit  
- Go：gin、cobra  
これらも 全部モジュールの集合体  
<br>
<br> 

## 言語ごとにモジュールの呼び方は異なるが、意味はほぼ同じ  
| 言語         | 呼び方             | 文法（読み込み）                          | 管理方法           |
| ---------- | --------------- | --------------------------------- | -------------- |
| Python     | モジュール・パッケージ     | `import xxx`                      | pip            |
| JavaScript | モジュール           | `import xxx from "xxx"`           | npm / yarn     |
| Java       | パッケージ           | `import package.class`            | Maven / Gradle |
| C#         | 名前空間（namespace） | `using System;`                   | NuGet          |
| Go         | パッケージ           | `import ("fmt")`                  | go modules     |
| PHP        | モジュール・ライブラリ     | `use Vendor\Package;` / `require` | Composer       |

<br>
<br> 


## Python モジュールの種類  
### 🍡標準モジュール（Python に最初から入っているもの）  
インストール不要で、すぐに使えるモジュール  

| モジュール          | 用途                   |
| -------------- | -------------------- |
| **math**       | 数学計算（平方根、三角関数など）     |
| **random**     | 乱数を作る                |
| **datetime**   | 日付や時刻を扱う             |
| **os**         | OS（フォルダ操作・環境変数など）と連携 |
| **sys**        | Python自体の動作を制御       |
| **json**       | JSONデータの読み書き         |
| **re**         | 正規表現（文字列の検索・置換）      |
| **statistics** | 統計（平均・中央値など）         |

<br> 

  
### 🍡外部モジュール（自分でインストールするもの）  
pip install でインストールして使うパッケージ  

| モジュール              | 分野      | 用途          |
| ------------------ | ------- | ----------- |
| **numpy**          | 数値計算    | 高速な計算処理     |
| **pandas**         | データ分析   | 表形式データの処理   |
| **matplotlib**     | グラフ描画   | グラフ作成       |
| **requests**       | Webアクセス | HTTP通信      |
| **Flask / Django** | Web開発   | Webアプリ作成    |
| **opencv-python**  | 画像処理    | 画像認識・編集     |
| **pyautogui**      | 自動化     | マウス・キーボード操作 |
