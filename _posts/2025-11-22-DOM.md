📘 1ページ目：DOMとは何か？／DOM要素の取得方法

🌐 DOMとは？
<p> <strong>DOM（Document Object Model）</strong> は <span style="color: #00ff00;">HTML を JavaScript が操作できるようにする仕組み</span> です。 </p>

HTML はブラウザによって読み込まれると、内部で
<strong>木構造のオブジェクト</strong> として扱われます。

その結果、JavaScript で次のような操作が可能になります。

テキストの変更

画像の差し替え

ボタンにイベントを付ける

CSS を動的に変更する

HTML 要素を追加・削除する

DOM は「JavaScript と HTML の橋渡し」と言えます。

🔍 DOM要素の取得方法

DOM を操作するためには、まず対象となる HTML の要素を取得する必要があります。
代表的な方法は以下の通りです。

🎯 getElementById() — ID を持つ要素を1つ取得
■ どんなときに使う？

ページ内に 1つだけ存在する要素を取得したいとき

■ できること

テキスト変更

CSS変更

イベント登録

値の取得

■ 書き方
const element = document.getElementById("demo");
console.log(element.innerText);

■ 特徴

もっとも高速で確実な取得方法

HTML では ID が一意であることが前提

🎯 getElementsByClassName() — クラス名で複数取得
■ どんなときに使う？

同じクラス名を持つ複数の要素をまとめて処理したい時

■ 書き方
const items = document.getElementsByClassName("foo");
for (const el of items) {
  console.log(el.innerText);
}

■ 特徴

HTMLCollection（ライブコレクション）を返す

DOMに変更があると内容も自動更新される

🎯 getElementsByTagName() — タグ名で複数取得
■ どんなときに使う？

ページ内のすべての <div> や <p> を取得したいとき

■ できること

全てのタグにスタイルを適用

一括で処理したい場合に便利

■ 書き方
const divs = document.getElementsByTagName("div");

■ 特徴

HTMLCollection を返す

タグ単位でまとまりを扱いたいときに使う

🌟 querySelector / querySelectorAll（現代JavaScriptの主流）
<p> <strong>CSS セレクタと同じ書き方で DOM を検索できる</strong> ため、 <span style="color:#00ff00;">現代 JavaScript ではもっともよく使われる方法</span> です。 </p> <p> ※ CSS を参照するのではなく、 <strong>CSS の書き方で DOM を検索しているだけ</strong> という点に注意 </p>
✔ querySelector — 条件に合う最初の1つだけ取得
■ どんなときに使う？

クラスでも ID でもタグでも柔軟に検索したい時

複雑なセレクタ（ul li:first-child など）を使いたい時

■ 書き方
const item = document.querySelector(".box");

■ 特徴

対応力が高い

1つだけ取得する

✔ querySelectorAll — 条件に合うすべての要素を取得
■ どんなときに使う？

.item のようなグループに対して同じ処理を行いたい時

■ 書き方
const items = document.querySelectorAll(".box");
items.forEach((el) => {
  el.style.color = "red";
});

■ 特徴

NodeList（静的コレクション）が返る

forEach が使えるので扱いやすい

📝 form要素の取得（document.forms）
■ どんなときに使う？

入力された値を取得したい時

フォーム送信前にチェックしたい時

■ 書き方
<form name="demoForm">
  <input id="title-id" name="title" type="text">
</form>

const form = document.forms.demoForm;
const title = form.title.value;
console.log(title);

■ 特徴

form.name でフォーム全体にアクセス

form.要素名 で input 要素を取得できる
