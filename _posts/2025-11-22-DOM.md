# DOMとは  
- Document Object Modelの略  
- JavaScriptからhtmlにアクセスする窓口     
- **html要素の値を取得したり変更したりする**  

JavaScript から「文字を変える」「画像を差し替える」「ボタンを押したら動く」などの操作ができるようにする仕組み  

## 値を取得する方法
 
```
１）id指定で取得 
HTML  
<div id="demo">
  demoというidが指定されていますよ
</div>

JavaScript  
const element = document.getElementById(demo)
console.log(element.innerText)
```  
```  
２）class指定で取得  
HTML  
<div class="foo">fooというクラスが指定されたdiv要素1</div>
<div class="foo">fooというクラスが指定されたdiv要素2</div>
<div class="foo">fooというクラスが指定されたdiv要素3</div>

JavaScript  
const classes = document.getElementsByClassName('foo')
for (const classNode of classes) {
    console.log(classNode.innerText)
}
```  
```  
３）form要素を取得    
HTML
<form> name="demoForm">
  <div>
    <label for="title-id">タイトル</label>
    <input id="title-id" name="title" type="text"/>>
  </div>
</form>


JavaScript
const forms = document.forms.demoForm  
// 値を取得したいinput要素のnameを取得   
const title = forms.title.value  
console.log(title)  
```
---  

element.style  
CSSと同じように、要素の色を変えたり、大きさを変えたり、位置を動かしたりすることができる  
要素.style.property（color、fontWeight、backgroundColor等） = "値"  

element.appendChild(element)  
新しい内容を追加したり、既存の内容を変更したりすることができる  

createElement(tag_type)  
新しいHTML要素（タグ）を作ることができる  
カッコ内には、作りたいHTML要素の名前を入れる  
const newElement = document.createElement(tag_type);

element.remove  
Webページの画像など、特定要素を取り除くための命令  
element.remove();  

element.removeChild()  
取り除きたい要素の「親」を見つけ、取り除きたい要素（子）を指定することができる  
親要素.removeChild(子要素);  

element.getAttribute()  
指定した要素の属性を取得する  

element.setAttribute()  
指定の要素に新しい属性を追加したり、指定の要素に存在する属性の値を変更したりできる  

element.removeAttribute()  
指定の要素の属性を削除できる  

input readonly  
HTMLで入力フォームを作成する際に使用される要素  
書き換えを禁止し読み出し専用にして、ユーザーが入力できないようにする
```
// 入力欄を読み取り専用にする
const inputElement = document.getElementById('input')
inputElement.setAttribute('id', 'inputId')
```

```
// ボタンの無効化
const inputElement = document.getElementById('input')
inputElement.setAttribute('id', 'inputId')
```
